ScoreKEEPER


ScoreKEEPER is is an application that keeps digital track of a player's score that can be used for a multitude of external games.

My Role

Using Google's Materialize as a CSS framework, I designed all of the visual elements and concentrated on the front-end of the app. Player scoring cards are generated dynamically on the "Play" page using JQuery.
